## Project Overview

`extras_fetcher` automates the discovery, downloading, conversion, and organization of supplemental video content (extras) for Jellyfin media libraries. It supports both movie and TV series libraries, pulling content from TMDB, Archive.org, YouTube, and TheTVDB, converting to x265/HEVC, and placing files into Jellyfin-compatible directory structures.

## Quick Reference

- **Binary:** `extras_fetcher`
- **Tech Stack:** Rust + Tokio + clap + reqwest + serde + yt-dlp + ffmpeg
- **Entry Point:** `src/main.rs`
- **Architecture Pattern:** Scan → Discover → Download → Convert → Organize
- **Config File:** `config.cfg` (JSON, auto-created on first run)
- **External Dependencies:** `yt-dlp` and `ffmpeg` must be in PATH

## Getting Started

### Prerequisites

- Rust toolchain (2024 edition) — https://rustup.rs
- `yt-dlp` in system PATH
- `ffmpeg` with HEVC support in system PATH
- TMDB API key (prompted on first run)

### Setup

```bash
cargo build --release
```

### Run Locally

```bash
# Process a movie library
./target/release/extras_fetcher /path/to/movies

# Process a TV series library
./target/release/extras_fetcher --series-only /path/to/tv

# Enable Season 0 specials (requires TVDB API key)
./target/release/extras_fetcher --series-only --specials /path/to/tv
```

### Run Tests

```bash
cargo test
```
